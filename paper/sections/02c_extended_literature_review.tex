\subsection{Analysis of the Research Landscape}

The current state of PQC literature heavily fragments into three distinct silos: (1) Core cryptographic community benchmarks (\texttt{m4} optimizations) which ignore communication overlays, (2) Protocol-level network benchmarks (TLS 1.3 surveys) which ignore bare-metal microcontroller cycle variance, and (3) Application-specific studies (e.g., V2X automotive) which test single configurations against specific hardware.

As illustrated in Table \ref{tab:lit_review}, our framework uniquely bridges these silos by evaluating the exact August 2024 FIPS standards concurrently against all three RFC 7228 memory boundaries ($<10$KB, $<50$KB, and $<250$KB), mapping strictly normalized execution times into a comprehensive 10-point TOPSIS decision matrix. Furthermore, while prior literature rarely combines dynamic CPU energy cycles with deterministic RF packet transmission wattage, our hybrid evaluation ($E_{total} = E_{cpu} + E_{tx}$) acts as the first definitively unified embedded energy baseline.
